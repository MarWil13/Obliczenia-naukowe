# Co zrobić z tym pakietem plików

Ten katalog zawiera propozycję plików, które można dodać do repozytorium `MarWil13/Obliczenia-naukowe`, żeby projekt wyglądał pełniej i lepiej odpowiadał wymaganiom projektu grupowego.

## Najważniejsze pliki

1. `.github/workflows/ci.yml` — automatyczne testy na GitHub Actions.
2. `docs/projekt_numeryczny.md` — opis założeń, algorytmów, złożoności, dokładności, wydajności i znaczenia praktycznego.
3. `docs/api.md` — dokumentacja eksportowanych funkcji.
4. `examples/basic_simulation.jl` — prosty przykład użycia pakietu.
5. `examples/compare_methods.jl` — porównanie metod numerycznych.
6. `notebooks/projekt_dyfuzja_zgodny_z_wymaganiami.ipynb` — notebook prezentacyjno-analityczny.
7. `README.md` — proponowana, czytelniejsza wersja pliku README.

## Uwaga

Jeżeli w repozytorium istnieje już `README.md`, ten plik zastąpi obecną wersję. Przed podmianą warto skopiować starą treść albo porównać zmiany.
