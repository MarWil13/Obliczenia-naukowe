# Figures

Ten katalog jest przeznaczony na wykresy wygenerowane z notebooka lub skryptów przykładowych, np.:

- profil temperatury w kilku chwilach czasu,
- porównanie metod Eulera jawnego, Eulera niejawnego i Cranka-Nicolsona,
- wykres błędu względem rozwiązania analitycznego,
- demonstrację niestabilności po przekroczeniu limitu CFL.

Pliki graficzne nie są wymagane do działania pakietu, ale pomagają w prezentacji projektu.
