# Notebooks

W tym katalogu można umieścić notebook prezentacyjno-analityczny projektu.

Proponowana nazwa:

```text
projekt_dyfuzja_zgodny_z_wymaganiami.ipynb
```

Notebook powinien zawierać uruchamialne przykłady, wykresy, demonstrację niestabilności metody jawnej, analizę dokładności i analizę wydajności.
